
# Game Event Managment and Participation

A fullstack event managment platform, for scheduling, activating and picking a winner from participants who upload the correct screenshots.





## Features

- Admin dashboard:
    - Participants moderation
    - Winner selection
    - Event and winner scheduling
    - Real time pending updates
    - Admin Authentication System
- Participants page
- Entries writedown in Neon
- Screenshot upload in Cloudinary


## Tech Stack

**Frontend:** React, Vite, TailwindCSS

**Backend:** FastAPI, Sqlalchemy

**Server:** Cloudinary, Neon


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

### Frontend:

`VITE_ADMIN_TOKEN`

### Backend

`DATABASE_URL`

`CLOUDINARY_CLOUD_NAME`

`CLOUDINARY_API_KEY`

`ADMIN_SUPER_SECRET_KEY`
